## 先把搬瓦工的线路类型分清楚

搬瓦工之所以价格差异这么大，核心就在"线路"两个字上。同样的1核1G内存、同样的20G硬盘，普通线路和CN2 GIA-E线路的年付价格能差出三倍不止。原因很简单：进中国方向的带宽成本不一样。

目前搬瓦工的产品线大致可以分成这么几档：

- **KVM 普通线路（KVM VPS）**：走的是美国本土机房（DC2、DC4、DC8、FMT等）的常规163骨干网，国际段稳定，但回中国方向没有优化，延迟较高、晚高峰容易丢包。这是搬瓦工最便宜的一档，年付起步价$49.99，适合纯海外业务、不在意国内访问速度的场景。
- **CN2 GIA-E 电商型（DC6/DC9等）**：走中国电信CN2 GIA专线（GT+GIA混合架构），三网回程优化，2.5Gbps大带宽起步，是搬瓦工的"主力招牌"。这一档适合建站、轻量代理、长期稳定项目。
- **CN2 GIA 单线路专享（HK / Tokyo / Osaka / Singapore）**：单一机房走CN2 GIA回程，延迟更低但流量相对小、价格更高。Tokyo、HK的入门款年付就到$899.99起，Osaka和Singapore因为有促销性质，年付能做到$499.99起。
- **DUBAI ECOMMERCE（迪拜机房）**：相对新的产品线，主打中东节点，但也包含DC6 CN2 GIA-E等回中国机房可选，性价比中等。

> 简单记一下：**KVM最便宜但回中国慢，CN2 GIA-E是综合性价比之王，HK/Tokyo/Singapore/Osaka是高端专属节点，Dubai是中东方向的新选择。**

## 年付套餐续费与升级的几个要点

买完之后还有几件事得知道：

**1. 续费价格不变**
搬瓦工的年付套餐是终身价，续费时按你最初购买的金额来，不会偷偷涨价。你当年用优惠码省了6.78%，续费也会自动继续省6.78%。

**2. 套餐可升级，不可降级**
KiwiVM控制面板里支持"Upgrade/Downgrade"，但实际操作中从高档降到低档基本不行，只能升。升级时按剩余时间比例补差价。

**3. 机房可迁移**
CN2 GIA-E、DUBAI ECOMMERCE 等多机房套餐支持在KiwiVM里自助切换机房。HK、Tokyo、Osaka、Singapore 这类单一机房套餐原则上不支持跨机房迁移——选之前想清楚。

**4. 流量超额按GB收费**
所有套餐流量都是月度结算，超出部分按官方公示的$1/GB左右收取（具体以官网为准）。如果业务经常超额，建议直接买大一档而不是按超额收费。

**5. 支付方式**
搬瓦工支持支付宝、PayPal、信用卡、银联等，国内用户买起来挺方便。
